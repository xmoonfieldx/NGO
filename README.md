# NGO

## Installation

### Clone the Repository

```
git clone git@github.com:xmoonfieldx/NGO.git
```

### For Client Directory

```
cd client
yarn install
```

### For Server Directory

```
cd server
yarn install
```

## Running the application

### For running the client locally

```
yarn run start
```

### For running the server locally

```
yarn run start
```
